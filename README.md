## Description

This is a NestJS APIs to provide Prices endpoints.
You can find the Swagger documentation at:

https://prices.murrayrothbot.com/swagger

## Installation

```bash
$ npm install
```

## Running

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Extras

```bash
# remove all deps and reinstall from scratch
$ npm run clean

# check for dependencies updates
$ npm run check
```
